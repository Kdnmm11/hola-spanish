Si 조건문과 가정법

실현 가능성에 따라 3가지 유형으로 나뉩니다.
각 유형별 조건절(si)과 귀결절의 시제 조합 공식이 핵심입니다.

핵심 요약

  • 1유형 (현실): si + 직설법 현재 → 직설법 현재/미래/명령.
  • 2유형 (가상): si + 접속법 과거 → 조건형.
  • 3유형 (후회): si + 접속법 과거완료 → 조건형 완료.

1. 제1유형: 실현 가능한 조건

현재나 미래에 충분히 일어날 수 있는 현실적인 상황을 가정합니다.

공식: Si + 직설법 현재, [현재/미래/명령]
현실적 가능성

충분히 일어날 수 있는 미래의 일이나 일반적인 습관을 말합니다. "만약 ~하면, ~할 것이다"라는 뜻입니다.

Si조건tengo직설법 현재tiempo,te ayudaré.직설법 미래

시간이 나면 (실제로 날 가능성 있음), 너를 도와줄게.

Si 뒤에는 절대 미래 시제를 쓰지 않습니다!

Si tendré tiempo (X)Si tengo tiempo (O)

2. 제2유형: 현재의 가상/희박한 조건

현재 사실과 반대되거나 실현 가능성이 매우 낮은 상상을 할 때 씁니다.

공식: Si + 접속법 과거, [조건형]
현재 사실의 반대

지금 현재 일어나지 않은 일을 상상합니다. 현재와 거리를 두기 위해 한 단계 과거 시제인 "접속법 과거"를 사용합니다.

Si조건fuera접속법 과거rico,compraría조건형una mansión.

내가 (지금) 부자라면 (실제론 아님), 대저택을 살 텐데.

3. 제3유형: 과거의 불가능한 조건

이미 지나간 과거 사실에 반대되는 가정을 하며, 주로 후회나 비판을 나타냅니다.

공식: Si + 접속법 과거완료, [조건형 완료]
과거 사실의 반대

이미 종료된 과거의 사건을 뒤집어 생각합니다. "했었더라면 ~했을 텐데"라는 후회나 비판의 뉘앙스입니다.

Si조건hubiera estudiado접속법 과거완료más,habría aprobado조건형 완료el examen.

공부를 더 했었더라면 (실제론 안 함), 시험에 합격했을 텐데.

4. 시제 조합 요약표

유형Si 조건절귀결절 (주절)용도
1유형 (실현 가능)직설법 현재현재 / 미래 / 명령실제 가능성
2유형 (현재 가상)접속법 과거조건형현재 반대 / 희박
3유형 (과거 후회)접속법 과거완료조건형 완료과거 반대 / 불가능
