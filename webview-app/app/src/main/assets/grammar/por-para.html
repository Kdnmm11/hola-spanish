전치사 I (por, para)

원인·경로(Por) vs 목적·결과(Para)의 4대 핵심 대조를 학습합니다.

핵심 요약

  • para: 화살표의 끝(목적지, 수혜자, 마감, 용도)을 향합니다.
  • por: 화살표의 시작이나 통로(이유, 수단, 경로, 기간)를 나타냅니다.
  • 관용구: por favor, por ejemplo 등 굳어진 표현을 익히는 것도 중요합니다.

1. 전치사 para의 용법 (목표와 마감)

동작이 지향하는 최종 지점이나 목적을 나타낼 때 사용합니다.

용법설명예시
목적~하기 위하여 (동사원형 결합)
Estudio para aprender español.스페인어를 배우기 위해 공부한다.
수혜자~를 위하여 (대상 지칭)
Este regalo es para ti.이 선물은 너를 위한 것이다.
목적지~를 향하여 (이동 방향)
Salgo para Madrid.마드리드를 향해 출발한다.
마감 기한~까지 (완료 시점)
La tarea es para mañana.숙제는 내일까지다.
의견~가 보기에는 (판단 기준)
Para mí, es difícil.내가 보기에는 어렵다.

2. 전치사 por의 용법 (원인과 경로)

동작이 일어나는 동기, 수단, 혹은 통과하는 경로를 나타낼 때 사용합니다.

용법설명예시
원인/이유~ 때문에 (동기)
Lo hago por amor.사랑 때문에 그것을 한다.
경로~를 통하여 (통과 지점)
Paso por el parque.공원을 지나간다.
기간~ 동안 (시간의 길이)
Estudio por dos horas.두 시간 동안 공부한다.
교환/가격~ 대신에 / ~의 가격으로
Te doy cinco euros por el libro.그 책의 대가로 5유로를 준다.
수단~로 / ~를 통해서 (통신/교통)
Hablo por teléfono.전화로 이야기한다.

3. 핵심 대조 요약

상황에 따른 두 전치사의 개념 차이를 한눈에 비교합니다.

비교 항목por (시작/과정)para (결과/목표)
장소경로 (통과, 근처)목적지 (도착점)
시간기간 (길이)마감 (기한)
인과원인 (동기)목적 (의도)
관계교환 (대신하여)수혜 (위하여)

4. 필수 관용구 (expresiones)

하나의 단어처럼 굳어져 사용되는 주요 표현들입니다.

por 계열

관용구의미예문
por favor부탁합니다
Un café, por favor.커피 한 잔 부탁해요.
por ejemplo예를 들어
Me gustan las frutas, por ejemplo, la manzana.나는 과일을 좋아해, 예를 들어 사과.
por fin마침내
¡Por fin has llegado!마침내 도착했구나!
por ciento퍼센트 (%)
El cien por ciento de los alumnos.학생의 100%.

para 계열

관용구의미예문
para siempre영원히
Te amaré para siempre.너를 영원히 사랑할 거야.
para qué무엇을 위해
¿Para qué sirve esto?이건 뭐에 쓰는 거야?
para variar웬일로/변화를 위해
Vamos al cine para variar.기분 전환 겸 영화 보러 가자.

기초 다지기 (práctica)

Q1

빈칸 채우기 (목적): estudio mucho ( ) ser médico.

Q2

빈칸 채우기 (이유): gracias ( ) tu ayuda.

Q3

틀린 문장 고치기 (세비야가 목적지일 때): el tren va ( ) sevilla.

Q4

빈칸 채우기 (수단): te llamo ( ) teléfono esta tarde.
